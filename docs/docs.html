Introduction

AckerJS is a lightweight, modular JavaScript framework designed to make modern web development simpler and faster. It provides essential utilities for DOM manipulation, HTTP requests, and data formatting without the bloat of larger frameworks.

Key features:

  • Lightweight: Zero dependencies and tiny bundle size.
  • Modular: Import only what you need.
  • TypeScript: Built with TypeScript for excellent developer experience.
  • Components: Simple, lightweight Web Component system.

Installation

You can install AckerJS via npm:

npm install ackerjs

Or use the CLI to scaffold a new project:

npx create-acker-app my-app

Basic Usage

AckerJS is modular, so you can import everything or just specific modules.

Import Everything

import { DOM, HTTP, Format } from 'ackerjs';

const el = DOM.select('#app');
HTTP.get('/api/data').then(data => console.log(data));

Import Specific Modules

import { select, create } from 'ackerjs/dom';
import { fetchJSON } from 'ackerjs/http';
import { formatDate } from 'ackerjs/format';

DOM Utilities

A collection of helper functions for selecting, creating, and manipulating DOM elements.

Selection

select(selector)

Select a single element.

const btn = select('#submit-btn');

selectAll(selector)

Select all matching elements as an array.

const items = selectAll('.item');
items.forEach(item => item.style.color = 'red');

Manipulation

create(tag, props)

Create a new element with properties.

const div = create('div', {
  className: 'container',
  id: 'main',
  style: { color: 'blue' },
  onclick: () => console.log('Clicked!')
});

append(parent, child)

Append a child element to a parent.

append(document.body, div);

remove(element)

Remove an element from the DOM.

remove(div);

empty(element)

Remove all child nodes from an element.

empty(listContainer);

clone(element, deep)

Clone an element.

const copy = clone(original, true);

Classes & Attributes

addClass(element, ...classes)

addClass(btn, 'active', 'primary');

removeClass(element, ...classes)

removeClass(btn, 'active');

toggleClass(element, className)

toggleClass(btn, 'hidden');

hasClass(element, className)

if (hasClass(btn, 'active')) { ... }

setAttributes(element, attributes)

setAttributes(input, { type: 'text', placeholder: 'Name' });

getAttribute(element, attribute)

const id = getAttribute(el, 'data-id');

Events

on(element, event, handler, options)

on(btn, 'click', handleClick);

off(element, event, handler, options)

off(btn, 'click', handleClick);

Traversal

matches(element, selector)

if (matches(target, '.btn')) { ... }

closest(element, selector)

const form = closest(input, 'form');

HTTP Utilities

Wrappers around the Fetch API with built-in error handling and JSON parsing.

Core Methods

fetchJSON(url, options)

Fetch JSON data from a URL.

const data = await fetchJSON('/api/users');

get(url, options)

Alias for fetchJSON (GET request).

post(url, data, options)

Send a POST request with JSON data.

const result = await post('/api/users', { name: 'Alice' });

put(url, data, options)

Send a PUT request.

await put('/api/users/1', { name: 'Alice Smith' });

patch(url, data, options)

Send a PATCH request.

del(url, options)

Send a DELETE request.

await del('/api/users/1');

Helpers

createFetch(baseURL, defaultOptions)

Create a configured fetch instance.

const api = createFetch('https://api.example.com', {
  headers: { 'Authorization': 'Bearer token' }
});
const users = await api.get('/users');

buildURL(baseURL, params)

Construct a URL with query parameters.

const url = buildURL('/search', { q: 'test', page: 1 });
// /search?q=test&page=1

Formatting Utilities

Helpers for formatting strings, numbers, and dates.

Date & Time

  • formatDate(date, locale)
  • formatDateTime(date, locale)
  • formatTime(date, locale)
  • formatRelativeTime(date, locale) - e.g., "2 hours ago"

Strings

  • capitalize(str)
  • capitalizeWords(str)
  • truncate(str, length, suffix)
  • slugify(str)
  • camelCase(str)
  • kebabCase(str)
  • snakeCase(str)
  • escapeHTML(str)

Numbers

  • formatNumber(num, decimals, locale)
  • formatCurrency(amount, currency, locale)
  • formatPercent(value, decimals, normalize, locale)
  • formatBytes(bytes, decimals)

Validation

  • isEmail(email)
  • isURL(url)

Component System

A lightweight wrapper around Web Components.

Defining a Component

import { component } from 'ackerjs';

component('user-card', (props) => {
  return `
    <div class="card">
      <h3>${props.name}</h3>
      <p>${props.role}</p>
    </div>
  `;
}, ['name', 'role']);

Using a Component

<user-card name="Alice" role="Admin"></user-card>

Props

Pass props as attributes. The component will re-render when observed attributes change.
