Introduction

AckerJS is a lightweight, modular JavaScript framework designed to make modern web development simpler and faster. It provides essential utilities for DOM manipulation, HTTP requests, and data formatting without the bloat of larger frameworks.

Installation

You can install AckerJS via npm:

npm install ackerjs

Or use the CLI to scaffold a new project:

npx create-acker-app my-app

Component System

AckerJS 1.0 introduces a lightweight component system based on Web Components.

Defining a Component

import { component } from 'ackerjs';

component('user-card', (props) => {
  return `
    <div class="card">
      <h3>${props.name}</h3>
      <p>${props.role}</p>
    </div>
  `;
}, ['name', 'role']);

Using a Component

<user-card name="Alice" role="Admin"></user-card>
