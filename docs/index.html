Quick Examples

DOM Manipulation

import { DOM } from 'ackerjs';

const button = DOM.create('button', {
  textContent: 'Click Me',
  className: 'btn-primary'
});

DOM.on(button, 'click', () => {
  alert('Button clicked!');
});

DOM.append(document.body, button);

HTTP Requests

import { HTTP } from 'ackerjs';

// Fetch data
const user = await HTTP.fetchJSON(
  'https://api.example.com/user/1'
);

// Post data
await HTTP.postJSON('/api/users', {
  name: 'John Doe',
  email: 'john@example.com'
});

Data Formatting

import { Format } from 'ackerjs';

// Dates
Format.formatDate(new Date());
// "11/16/2025"

// Currency
Format.formatCurrency(1234.56);
// "$1,234.56"

// Strings
Format.slugify('Hello World!');
// "hello-world"

Documentation

DOM Helpers

  • select(selector) - Select a single element
  • selectAll(selector) - Select multiple elements
  • create(tag, props) - Create a new element
  • append(parent, child) - Append child to parent
  • remove(element) - Remove element from DOM
  • addClass/removeClass - Manage CSS classes
  • on/off - Event listeners
  • attr/data - Attributes and data

HTTP Utilities

  • fetchJSON(url) - GET JSON data
  • postJSON(url, data) - POST with JSON
  • putJSON(url, data) - PUT with JSON
  • deleteJSON(url) - DELETE request
  • get/post/put/delete - Generic methods
  • createAPI(baseURL) - API instance

Format Utilities

  • formatDate/DateTime/Time - Date formatting
  • formatNumber/Currency - Number formatting
  • capitalize/capitalizeWords - Text casing
  • truncate(str, length) - Truncate strings
  • slugify(str) - URL-friendly slugs
  • camelCase/kebabCase - Case conversion
